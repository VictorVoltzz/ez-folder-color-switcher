# ez-folder-color-switcher
Folder color switch extension for Nemo, Linux mint default file manager, this addon focus on on making it easy for users to be able to change folder color of other themes besides mint default ones, by simply creating a folder named "folder-color" in the root of your icon theme, then placing the folder variations inside.
inside the "folder-color" folder, you will place folder icons of your theme with different colors, naming them like:
folder-blue
folder-orange
folder-green
folder-pink
folder-gray
etc.
place the extension in /usr/share/nemo-python/extensions
restart nemo and enable it in nemo preferences, if not enabled on the get go, this will enable users to easily manage different folder colors of their downloaded icon themes.
